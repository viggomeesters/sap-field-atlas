# Support

SAP Field Atlas is a public, best-effort knowledge base. It is not an official SAP support channel and does not provide customer/project-specific advice.

## Where to ask

Use GitHub issues for:

- adding generic SAP objects;
- correcting a transaction/table/field/template mapping;
- reporting validation or package problems;
- improving source references or confidence labels.

Before opening an issue, check:

- `README.md`
- `CONTRIBUTING.md`
- `docs/source-confidence-policy.md`
- existing issues and release notes

## What not to share

Do not include:

- customer or client names;
- screenshots, exports, dumps, logs, or table extracts from SAP systems;
- internal URLs, tickets, project IDs, credentials, or proprietary mappings;
- copied proprietary SAP documentation.

If a support request requires sensitive context, it is out of scope for this public repo.

## Maintainer response expectations

This repository is maintained on a best-effort basis. There is no guaranteed response time or SLA.
