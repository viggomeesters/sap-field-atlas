# SAP Field Atlas v0.1.0

Initial public release of SAP Field Atlas: an agent-ready SAP data migration knowledge base using validated YAML contracts.

## Included

- Data contracts for transactions, Fiori apps, tables, fields, migration templates, domains/value sources, and relationships.
- Starter knowledge for `SE16N`, `SE11`, `MARA`, `MARA-MATNR`, and `T001-BUKRS`.
- Validation CLI: `sap-field-atlas validate` and `sap-field-atlas audit-completeness`.
- Agent explanation contract and example outputs.
- Public safety, source confidence, and contribution policy.
- Repository-complete guardrails: `AGENTS.md`, `Makefile`, repository guard, PR template, security doc, architecture/roadmap docs, professional README, hero asset, contributors file, and release package artifacts.

## Package artifacts

- `sap_field_atlas-0.1.0-py3-none-any.whl`
- `sap_field_atlas-0.1.0.tar.gz`

## Validation

Release built after:

```bash
make check
make package
```

No GitHub Actions workflow is included by default; local `make check` is the canonical gate.
