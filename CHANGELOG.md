# Changelog

All notable changes to SAP Field Atlas are documented here.

This project follows a lightweight semantic-versioning convention while it is pre-1.0: breaking contract changes may happen in minor releases, but each release should clearly describe data/schema/CLI impact.

## [0.1.0] - 2026-06-22

### Added

- Initial public SAP Field Atlas repository.
- YAML contracts for transactions, Fiori apps, tables, fields, migration templates, domains/value sources, and relationships.
- Starter knowledge for `SE16N`, `SE11`, `MARA`, `MARA-MATNR`, and `T001-BUKRS`.
- Validation CLI: `sap-field-atlas validate` and `sap-field-atlas audit-completeness`.
- Agent explanation contract and example outputs.
- Public-data safety policy, source-confidence policy, and contribution templates.
- Professional public repo surface: README hero, package docs, contributors, support, security, notice, roadmap, architecture docs, and release artifacts.
